    프로젝트 개요
    자동 개폐식 스마트 배수구 시스템
    강수량 및 습도 데이터를 실시간으로 수집하여 배수 그레이팅의 개폐 각도를 자동으로 제어하는 스마트 시스템입니다. BME280 환경 센서로 습도를 측정하고 블루투스 통신으로 구동부에 명령을 전달하며, 웹 페이지를 통한 수동 제어도 지원합니다.
    IoT Arduino Bluetooth 서보모터 웹 제어 종합설계 A
    시스템 구성도
    🌡️ BME280 온도·습도·기압
    →
    🖥️ Master Arduino 제어 로직 + LCD
    →
    📡 HC-05 Master 블루투스 송신
    ↓
    블루투스 무선 통신 (9600 baud)
    ↓
    📡 HC-05 Slave 블루투스 수신
    →
    🖥️ Slave Arduino 명령 처리 + LED
    →
    ⚙️ PCA9685 서보 드라이버
    →
    🔧 MG996R ×2 그레이팅 구동
    💻 웹 페이지 수동 제어 (USB Serial)

    작동 원리
    1
    환경 감지
    BME280 센서가 2초마다 온도·습도·기압을 측정합니다.
    2
    모드 판단
    습도값을 기준으로 3단계 모드를 자동 결정합니다.
    3
    블루투스 전송
    Master HC-05 → Slave HC-05로 명령 문자(0/1/2)를 무선 전송합니다.
    4
    서보 구동
    Slave Arduino가 명령을 수신하여 PCA9685를 통해 서보모터 2개를 부드럽게 회전시킵니다.
    습도 기준 모드 서보 각도 LED 색상
    ≤ 78.5% 비 없음 0° (닫힘) 🔴 빨강
    78.5% ~ 91.5% 약한 비 45° 개방 🟢 초록
    ≥ 91.5% 강한 비 90° 개방 🔵 파랑
